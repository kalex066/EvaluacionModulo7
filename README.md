Para finalizar el Modulo 7 se desarrollo una aplicación web con Django que permite gestionar productos, categorías y etiquetas de forma eficiente, con las siguientes caracteristicas:

- Autenticación de usuarios con protección de vistas mediante `@login_required`
- CRUD completo para productos, categorías y etiquetas
- Paginación con Bootstrap en todos los listados
- Mensajes de éxito y validación visual con `django.contrib.messages`
- Asociación de productos con categorías y etiquetas (ManyToMany y ForeignKey)
- Panel de administración personalizado para gestión avanzada

Tecnologías utilizadas
- **Python 3.11+**
- **Django 4.x**
- **Bootstrap 5**
- HTML5, CSS3
- MySQL
